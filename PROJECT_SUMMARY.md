# Project Summary

## Title
Explainable Ensemble Deep Learning Pipeline for Lung Cancer Histopathology Classification

## Summary
This project applies deep learning and explainable AI to classify lung histopathology images into adenocarcinoma, squamous cell carcinoma, and benign tissue.

The project uses transfer learning with ResNet-50, DenseNet-121, and EfficientNet-B0. A soft-voting ensemble combines model predictions to improve robustness. Grad-CAM is used to generate visual explanations, helping identify the image regions that influenced each model prediction.

- Ensemble learning
- Explainable AI
- Grad-CAM visualisation
- Model evaluation
- Error analysis

Python, PyTorch, Torchvision, NumPy, Pandas, Matplotlib, Scikit-learn, OpenCV, Grad-CAM, Google Colab.
